1) First we need to install the EKSCTL:
    curl --silent --location "https://github.com/weaveworks/eksctl/releases/latest/download/eksctl_$(uname -s)_amd64.tar.gz" | tar xz -C /tmp

Move the extracted binary to /usr/local/bin.# Anyuser can access if we it to localbin
sudo mv /tmp/eksctl /usr/local/bin



Test that your installation was successful with the following command. You must have eksctl 0.34.0 version or later.
eksctl version
